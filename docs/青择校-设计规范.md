# 青择校设计规范

版本：v0.1  
日期：2026-05-17  
设计载体：Figma / Stitch  
实现载体：微信小程序 Taro 4  
关键词：极简、克制、高级感、留白、半透明、液态玻璃、白色背景、青色主题、轻量、无阅读压力

## 1. 设计定位

青择校的视觉目标不是“招生平台”，而是“可信赖的择校查询工具”。设计必须像现代互联网工具产品，克制、轻、快、清晰。

目标感受：

- 像夸克 APP 一样直接。
- 像微信小程序官方规范一样轻量。
- 像 Apple 产品一样重视留白和细节。
- 像抖音团购详情页一样信息快速可扫。

禁止感受：

- 电商促销感
- 招生广告感
- 信息门户感
- 传统教育平台感
- 花哨运营活动页感

## 2. 品牌名称

主名称：青择校  
英文标识：Qing School Guide  
品牌短语：搜学校 搜专业

Logo 方向：

- 极简青色几何符号。
- 可用“青”字抽象化，也可用搜索镜 + 校园轮廓组合。
- 不使用盾牌、奖章、榜单等容易产生营销背书感的图形。

## 3. 色彩系统

主色：

| Token | 色值 | 用途 |
| --- | --- | --- |
| `--color-primary` | `#14C8C8` | 搜索按钮、重点数字、评分 |
| `--color-primary-deep` | `#07999B` | 标题渐变、深色强调 |
| `--color-primary-soft` | `rgba(20, 200, 200, 0.10)` | 浅青背景 |
| `--color-glass` | `rgba(20, 200, 200, 0.16)` | 液态玻璃卡片 |
| `--color-glass-border` | `rgba(20, 200, 200, 0.24)` | 玻璃描边 |

中性色：

| Token | 色值 | 用途 |
| --- | --- | --- |
| `--color-bg` | `#FFFFFF` | 页面背景 |
| `--color-text` | `#172326` | 主文本 |
| `--color-text-secondary` | `#6B7A7D` | 次文本 |
| `--color-text-muted` | `#A6B0B3` | 弱文本 |
| `--color-line` | `rgba(23, 35, 38, 0.08)` | 分割线 |

使用规则：

- 页面背景必须是纯白。
- 青色只用于核心交互、重点数据和品牌表达。
- 不使用大面积深色背景。
- 不做紫蓝大渐变、橙色营销色、红色促销色。

## 4. 字体系统

字体策略：

- 小程序内使用系统字体。
- 标题使用较高字重，模拟阿里巴巴普惠体的现代几何感。
- 不使用花体、手写体、过度装饰字体。

字号：

| 场景 | 字号 |
| --- | --- |
| 首页主标题 | 34-40px |
| 页面主标题 | 24-28px |
| 模块标题 | 18-22px |
| 卡片标题 | 16-18px |
| 正文 | 14-16px |
| 辅助信息 | 12-13px |

文字规则：

- 首页主标题可以大。
- 卡片和模块内标题必须克制。
- 不允许正文大段堆叠。
- 每个卡片优先展示 2 到 4 个关键信息。

## 5. 间距系统

基础单位：4px。

常用间距：

| Token | 值 |
| --- | --- |
| `--space-1` | 4px |
| `--space-2` | 8px |
| `--space-3` | 12px |
| `--space-4` | 16px |
| `--space-5` | 20px |
| `--space-6` | 24px |
| `--space-8` | 32px |
| `--space-10` | 40px |

规则：

- 首页必须有大量留白。
- 列表页信息密度适中，一屏至少 3 张卡。
- 详情页模块之间留白明确，不做密集分割线。

## 6. 圆角系统

| 场景 | 圆角 |
| --- | --- |
| 搜索框 | 26px |
| 学校卡片 | 22px |
| AI 解读卡片 | 28px |
| 图片 | 16px |
| 标签 | 999px |
| 底部打分按钮 | 999px |

规则：

- 圆角要现代，但不能儿童化。
- 卡片圆角统一，避免每个模块不同。

## 7. 阴影系统

搜索框：

```scss
box-shadow: 0 12px 30px rgba(20, 200, 200, 0.20);
```

学校卡片：

```scss
box-shadow: 0 10px 24px rgba(7, 153, 155, 0.08);
```

AI 卡片：

```scss
box-shadow: 0 18px 42px rgba(20, 200, 200, 0.18);
```

规则：

- 阴影必须柔和。
- 不使用黑色重阴影。
- 首页热词阴影弱于搜索框。

## 8. 液态玻璃风格

使用范围：

- 学校标签
- AI 解读报告
- 底部打分按钮

视觉参数：

```scss
background: linear-gradient(
  135deg,
  rgba(20, 200, 200, 0.20),
  rgba(255, 255, 255, 0.72)
);
border: 1px solid rgba(20, 200, 200, 0.24);
backdrop-filter: blur(18px);
```

规则：

- 玻璃效果只做重点模块。
- 不把所有卡片都做成玻璃。
- 保持文字清晰，不牺牲可读性。

## 9. 图标规则

图标用途：

- 搜索
- 地区
- 学费
- 专业
- 资质
- 类型
- 评分
- 学校数据
- 箭头入口

规则：

- 优先线性图标。
- 线宽统一。
- 不使用彩色复杂插画。
- 能用图标表达的辅助文字尽量减少。

## 10. 页面设计规范

### 10.1 首页

结构：

```text
顶部安全留白
主标题：搜学校 搜专业
搜索框
热词标签，不超过 5 个
大面积留白
```

要求：

- 无底部导航。
- 无信息流。
- 无推荐学校。
- 无招生广告。
- 主体位于页面中上区域。
- 搜索框高度偏高，按钮为青色。

### 10.2 列表页

结构：

```text
顶部安全留白
搜索框
筛选栏
学校卡片列表
```

学校卡片内容：

- 学校图片
- 学校名称
- 标签
- 评分
- 查看详情文字入口

要求：

- 一屏至少 3 张学校卡。
- “查看详情”不是按钮。
- 筛选项全部可取消。

### 10.3 详情页

结构：

```text
图片组
学校名称
学校标签
AI 解读报告
关键数据
热门专业
多维评分
底部打分按钮
```

要求：

- 图片高度不能过高。
- 一行显示 2 到 3 张图片。
- AI 解读报告必须是页面视觉重点。
- 关键数据全部使用主题色。
- 多维评分包含 A+ 体系和五边形评分图。

## 11. 动效规范

动效原则：

- 快速
- 轻
- 不抢注意力
- 不装饰性炫技

建议：

- 页面进入：轻微上移 + 淡入，180ms。
- 筛选弹层：底部上滑，220ms。
- 卡片点击：缩放到 0.985，80ms 后恢复。
- 标签选中：青色淡入，不做弹跳。

## 12. 空状态与加载态

加载页：

- Logo 居中。
- 背景纯白。
- 不显示复杂文案。

列表加载：

- 使用轻量骨架屏。
- 骨架块圆角与真实卡片一致。

无结果：

- 简短提示：“没有找到符合条件的学校”。
- 提供清空筛选入口。
- 不推荐其他学校。

## 14. 视觉验收清单

- 首页是否只剩搜索、热词和留白。
- 是否保留微信胶囊安全区域。
- 是否没有传统标题栏。
- 是否没有底部导航。
- 青色是否是唯一主色。
- 卡片是否轻，不像电商商品卡。
- 详情页 AI 卡片是否足够突出。
- 标签是否有半透明玻璃质感。
- 是否没有招生营销话术。
- 是否没有大段文字压迫感。
