# 青择校

一个极简、轻量、高信任感的陕西职业学校查询与 AI 择校参考微信小程序。

## 项目定位

青择校面向初中毕业生、高中毕业生和家长，聚焦学校真实性、专业信息、学费补助、校园关键数据和 AI 解读。产品只保留搜索、筛选、列表、详情四类核心能力，避免传统招生平台的信息堆积和营销话术。

## 技术栈

- 微信小程序
- Taro 4
- React
- TypeScript
- SCSS
- pnpm
